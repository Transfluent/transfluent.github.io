cURL Example to retrieve a file

curl "https://transfluent.com/v2/file/read/?identifier=test.json&language=en-gb&token=YOUR-TOKEN-GOES-HERE" --verbose

Successfully retrieved file (HTTP200, Content-Description: File Transfer; Content-Disposition: attachment; filename="teston_en-gb.js"):

{% highlight json %} {"test":"This is just a small test"} {% endhighlight %}

Failed to retrieve the file (HTTP400, Content-Type: application/json):

{% highlight json %} { "status": "ERROR", "error": { "type": "ELanguageNotSupportedException", "message": "Language is not supported: af-az" } } {% endhighlight %}
