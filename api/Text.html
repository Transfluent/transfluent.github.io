cURL Example to save a source text

curl -d '{"text_id":"sample-key","group_id":"SampleProject","language":"en-gb","text":"This is an example of source text","token":"YOUR-TOKEN-GOES-HERE"}' --header "Content-Type: application/json" https://transfluent.com/v2/text/ | python -m json.tool

{% highlight json %} { "response": true, "status": "OK" } {% endhighlight %}

{% highlight json %} { "error": { "message": "Token is invalid or has expired!", "type": "EBackendSecurityViolation" }, "response": "Token is invalid or has expired!", "status": "ERROR" } {% endhighlight %}
